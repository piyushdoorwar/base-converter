Quick Guide

Type in any box to update the others instantly. Separate numbers with spaces or commas.

ASCII / Text
ASCII is a standard table that maps characters to numbers. Text input shows readable characters.
Binary (Base 2)
Used for raw bits in computers. Enter as 8-bit groups like 01001000.
Hexadecimal (Base 16)
Compact for bytes. Common in programming, colors, memory addresses. Use 2-digit groups.
Decimal (Base 10)
Human-friendly numbers. ASCII codes are typically shown in decimal.
Octal (Base 8)
Legacy and Unix permissions. Use 3-digit groups for bytes.
